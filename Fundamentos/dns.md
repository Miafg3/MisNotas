## 🌐 Domain Name System (DNS)

**DNS (Sistema de Nombres de Dominio)** es el sistema encargado de traducir los nombres de dominio en direcciones IP.

Actúa como un traductor entre el lenguaje humano y el lenguaje de las máquinas.

Relaciona esto:

```
google.com
```

Con esto:

```
142.250.72.14
```

Gracias a los DNS no necesitamos memorizar **direcciones IP** para navegar en internet.

## 🔄 Flujo DNS simplificado

- 1️⃣ Escribes un dominio como `google.com` en el navegador.
- 2️⃣ Tu dispositivo consulta a un **servidor DNS**.
- 3️⃣ El servidor DNS busca la IP asociada al dominio.
- 4️⃣ Devuelve la dirección IP encontrada.
- 5️⃣ El navegador usa esa IP para conectarse al servidor web.

Todo esto ocurre en milisegundos.
